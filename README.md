# Pong Simples

Pequeno jogo Pong feito com HTML, CSS e JavaScript.

Como rodar localmente:
1. Clone ou copie os arquivos para uma pasta.
2. Abra `index.html` no navegador (duplo clique) ou rode um servidor estático (recomendado) por exemplo:
   - Com Python 3: `python -m http.server 8000`
   - Com Node (serve): `npx serve .`
3. Acesse `http://localhost:8000` (se estiver usando servidor).

Controles:
- Mover mouse sobre o canvas para controlar o paddle esquerdo.
- Ou usar as teclas ↑ e ↓.
- Espaço = pausar/retomar.
- R = reiniciar.
- Clique no canvas para iniciar/pausar.
